Go to Dashboard » Pages » Add New

  1. Provide the page title

  2. Select the appropriate page template for doctors. There are 3 templates for doctors page.

    1. Doctors 2 Columns Template
    2. Doctors 3 Columns Template
    3. Doctors 4 Columns Template

  3. Enter some text contents if you want to.

  4. Publish the page once it is ready.

  5. After publishing, you need to add the newly created page in menu from Dashboard » Appearance » Menus
