    Sticky Header

    To enable sticky header go to Dashboard » Theme Options » General as pointed in image below.

    Look for the Sticky Header option in this section. Enable it and Save Changes.

    Boxed Layout

    To enable boxed layout of MedicalPress Theme please navigate to Dashboard » Theme Options » General as pointed in image below.

    Look for the Boxed Layout option in this section. Enable it and Save Changes.

    Translating The Theme

    This theme is translation ready but the support of RTL languages is not available for now.

    1. For translation to only one language please visit This Knowledge Base Article
    2. For translation to multiple languages you can use WPML Plugin
